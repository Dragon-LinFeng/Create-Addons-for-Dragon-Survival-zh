# 参考其他附属指南

1. **确认版权**：在解包和参考任何模组之前，请务必确认该模组的许可证（License）是否允许你查看其源代码或进行反编译学习。尊重他人的知识产权是开发者的基本素养。

2. **复制模组文件**：将你下载的龙之生存附属模组（`.jar` 文件）复制到之前规划好的参考目录中：

   ```
   D:/Programming/MinecraftMod/Dragonsurvival/Addon-Example/
   ```

3. **解压模组内容**：在 `Addon-Example` 目录下为这个模组创建一个新的文件夹（以模组名称命名），然后将模组 `.jar` 文件中的所有内容解压到这个文件夹中。

4. **链接补全核心**：
   - 将`D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/`中的 `spyglass.json` 文件复制到这个文件夹根目录中。
   - 修改以下配置（**请将路径替换为你自己的实际路径**），让 Spyglass 能够链接之前安装的补全项目：
     ```json
     {
         "env": {
             "useFilePolling": true,
             "dependencies": [
                 "@vanilla-mcdoc",
                 "file:///D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh"
             ],
             "mcmetaSummaryOverrides": {
                 "registries": {
                     "path": "file:///D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/spyglass/registries.json",
                     "replace": false
                 },
                 "commands": {
                     "path": "file:///D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/spyglass/commands.json",
                     "replace": false
                 }
             }
             // 其他配置
         }
     }
     ```

5. 用VSCode打开这个配置好的参考项目即可
