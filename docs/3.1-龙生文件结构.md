# 龙之生存数据包结构速查

该部分是根据经验编写的不保证100%正确

## 目录结构速览

```
💾数据根目录
 ├─📂assets
 │  └─📂dragonsurvival
 │     ├─📂custom_soul_icons - 自定义龙魂图标
 │     ├─📂skin - 皮肤组件
 │     │ ├─📂default_parts - 每个成长阶段的默认组件列表
 │     │ └─📂parts - 皮肤组件定义
 │     │    ├─📂base   - 基础
 │     │    ├─📂bottom - 底部
 │     │    ├─📂claws  - 爪子
 │     │    ├─📂extra  - 额外
 │     │    ├─📂eyes   - 眼睛
 │     │    ├─📂horns  - 角
 │     │    ├─📂magic  - 魔法
 │     │    ├─📂spikes - 刺
 │     │    └─📂teeth  - 牙齿
 │     ├─📂textures - 贴图
 │     │  ├─📂ability_effect - 技能实现的自定义效果图标
 │     │  ├─📂dragon
 │     │  │  ├─📂<物种id> - 物种选择界面的展示皮肤贴图
 │     │  │  └─📂custom  - 皮肤组件贴图
 │     │  ├─📂entity
 │     │  │  └─📂projectiles - 自定义弹射物贴图
 │     │  └─📂gui
 │     │     ├─📂custom
 │     │     │  ├─📂altar        - 物种旗帜
 │     │     │  ├─📂body         - 身体类型图标
 │     │     │  ├─📂casting_bars - 施法状态
 │     │     │  ├─📂food_icons   - 食物图标
 │     │     │  ├─📂mana_icons   - 魔力图标
 │     │     │  ├─📂stage        - 成长阶段图标
 │     │     │  └─📂supply_icons - 缺陷进度图标
 │     │     └─📂sprites
 │     │        ├─📂abilities - 技能图标
 │     │        └─📂penalties - 缺陷图标
 │     └─📂[其他Minecraft原版资源包内容]
 └─📂data
    └─📂dragonsurvival
       ├─📂data_maps
       │  └─📂dragonsurvival
       │     ├─📂dragon_body
       │     │  └─📄body_icons.json - 身体类型资源映射
       │     └─📂dragon_species
       │        ├─📄diet_entries.json       - 饮食条目
       │        ├─📄dragon_beacon_data.json - 龙之信标
       │        ├─📄end_platforms.json      - 末地平台
       │        └─📄stage_resources.json    - 成长阶段资源映射
       ├─📂dragonsurvival
       │  ├─📂dragon_ability  - 定义技能
       │  ├─📂dragon_penalty  - 定义缺陷
       │  ├─📂dragon_species  - 定义物种
       │  └─📂projectile_data - 定义自定义弹射物
       ├─📂structure
       │  └─📂end_spawn_platforms - 末地平台结构
       ├─📂tags
       │  └─📂dragonsurvival
       │     ├─📂dragon_ability
       │     │  ├─📄<物种id>.json - 将技能注册至物种
       │     │  └─📄order.json    - 对注册的技能排序
       │     ├─📂dragon_penalty
       │     │  └─📄<物种id>.json - 将缺陷注册至物种
       │     └─📂dragon_species
       │        ├─📄all.json      - 注册物种
       │        ├─📄<物种id>.json - 你自己的物种
       │        └─📄order.json    - 物种在物种选择界面的排序
       └─📂[其他Minecraft原版数据包内容]
```
