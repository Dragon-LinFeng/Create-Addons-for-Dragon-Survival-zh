# 基础环境部署指南

## 1. 准备目录结构

一个好的目录结构能带来更好的编码心情和更高的效率。以下是我个人使用的结构，供你参考：

```
💾D:/
 └─📂Programming/          // 代码项目的根目录
    ├─📂McDoc/              // 存放数据包补全项目
    │  └─📂dragonsurvival-mcdoc-completion-zh/ // 龙之生存补全核心
    └─📂MinecraftMod/        // 存放所有 Minecraft 相关模组
       └─📂Dragonsurvival/   // 龙之生存专属目录
          ├─📂Addon/          // 存放你的附属模组项目
          ├─📂Addon-Example/  // 存放用于参考的其他附属模组（解包后）
          └─📂Dragonsurvival/ // 存放龙之生存模组源代码（用于参考代码逻辑）
```

## 2. 安装 Java 21

本教程针对 Minecraft 1.21.1，因此需要 Java 21 开发环境。

- **下载地址**：[Oracle JDK 21 下载页](https://www.oracle.com/cn/java/technologies/downloads)
- **Windows 直链**：[jdk-21_windows-x64_bin.exe](https://download.oracle.com/java/21/latest/jdk-21_windows-x64_bin.exe)
- **安装提示**：如果没有特殊需求，建议保持默认安装路径，以便其他程序自动识别。

## 3. 安装 IntelliJ IDEA

IDEA 用于编写 Java 代码、运行项目以及将模组打包成 `.jar` 文件。

- **下载地址**：[IntelliJ IDEA 下载页](https://www.jetbrains.com/zh-cn/idea/download/other/) （推荐使用最新的社区版，即 Community Edition）

### 安装插件

| 名称                  | 插件 ID        | 作用               |
| --------------------- | -------------- | ------------------ |
| Minecraft Development | `minecraft-dev` | 快速创建和开发 Minecraft 模组 |

## 4. 安装 Visual Studio Code

1.21.1 的龙之生存已转向数据驱动，因此需要一个优秀的数据包编辑器。

- **下载地址**：[VSCode 下载页](https://code.visualstudio.com/)

**⚠️ 安装时请务必勾选以下两项**：

- [x] 将“通过 Code 打开”操作添加到 Windows 资源管理器目录上下文菜单

## 5. 配置 dragonsurvival-mcdoc-completion-zh 补全项目

这是一个为龙之生存模组提供数据包智能补全的关键项目。[项目地址](https://github.com/Dragon-LinFeng/dragonsurvival-mcdoc-completion-zh)。

### 5.1 部署核心

编写附属时需要频繁引用龙之生存 Mod 的资源，因此我们需要一份解压后的 Mod 作为补全核心。

1. 将龙之生存 Mod 文件（`.jar`）中的 `data`、`assets` 文件夹以及 `pack.mcmeta` 文件解压到一个纯英文路径下（例如：`D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/`）。
2. 从项目的 [Releases](https://github.com/Dragon-LinFeng/dragonsurvival-mcdoc-completion-zh/releases) 页面下载最新版本的补全文件，并将所有内容解压到上一步的目录中，覆盖同名文件夹。

### 5.2 检查安装

1. 用 VSCode 打开核心文件夹 (`D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/`)。
2. 打开 `data/dragonsurvival/dragon_ability/` 目录下的任意一个 `.json` 文件。
3. 将鼠标悬停在任意键上（例如 `"actions"`），如果出现悬浮注释，则说明安装成功。
4. 如果没有出现，请参考项目的[故障排除指南](https://github.com/Dragon-LinFeng/dragonsurvival-mcdoc-completion-zh/blob/master/docs/TROUBLESHOOTING.md)。

### 5.3 将补全核心加入 VSCode 项目管理器

1. 在 VSCode 中打开核心文件夹 (`D:/Programming/McDoc/dragonsurvival-mcdoc-completion-zh/`)。
2. 在左侧活动栏中找到「项目管理器 (Project Manager)」图标，点击「FAVORITES」区域旁边的储存图标（💾），将其保存为一个项目。
3. 之后，你就可以随时在 VSCode 中通过项目管理器快速访问这个补全核心，查阅龙之生存的原始数据了。
