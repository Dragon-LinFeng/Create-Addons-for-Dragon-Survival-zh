## 1. 在开始之前

在动手之前，你需要了解一个关键信息：虽然 Minecraft 的 Wiki 和教程大多是针对最新版本的，但我们的目标模组——龙之生存——目前停留在 **2024年10月22日** 之前发布的 1.21.1 版本。因此，在查找资料时，需要刻意寻找对应时间线的内容。

你可以在 Minecraft Wiki 页面右上角的“**查看历史**”中，使用“**过滤历史修订版本**”功能，将截止日期设定在目标版本之后（例如 1.21.2 版本的发布日期），以找到最合适的资料。

> **参考资料示例**：
> 1.21.1 时间线上的原版数据包介绍： [https://zh.minecraft.wiki/w/%E6%95%B0%E6%8D%AE%E5%8C%85?oldid=985861](https://zh.minecraft.wiki/w/%E6%95%B0%E6%8D%AE%E5%8C%85?oldid=985861)
